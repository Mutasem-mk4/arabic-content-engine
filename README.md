# Muharrir (محرّر) ⚡ Arabic Content Engine

Muharrir is a premium, AI-powered content repurposing engine designed to transform Arabic long-form content (articles, scripts, posts) into viral short-form assets across multiple dialects and platforms.

## ✨ Features

- **Multi-Dialect Localization:** Support for Gulf (خليجي), Egyptian (مصري), Levantine (شامي), and Maghrebi (مغربي).
- **Platform Specific:** Tailored outputs for TikTok Hooks, Twitter (X), Instagram, and LinkedIn.
- **Deep Cultural Reasoning:** Powered by Claude 3.5/3.7 with extended thinking to ensure cultural nuance and current slang (2024-2026).
- **Privacy First:** BYOK (Bring Your Own Key) model—your API key is stored locally in your browser.
- **Premium UI:** Dark mode with Arabic-inspired gold accents and full RTL support.

## 🚀 Getting Started

1. Clone this repository.
2. Open `index.html` in any modern web browser.
3. Enter your Anthropic API Key.
4. Paste your content and start generating!

## 🛠 Tech Stack

- **React** (via CDN)
- **Tailwind CSS** (via CDN)
- **Anthropic API** (Claude SDK)
